# FreeTP.org Source for Hydra Launcher

A lightweight JSON metadata source containing magnet links for FreeTP.org repacks, optimized for Hydra Launcher.

## How to Use

1. Copy the raw link to the JSON file:
   `https://raw.githubusercontent.com/GigaPich/testing/refs/heads/main/FreeTP_Repacks.json`
2. Open **Hydra Launcher**.
3. Go to **Settings** > **Download Sources** > **Add Source**.
4. Paste the link and click **Add**.
